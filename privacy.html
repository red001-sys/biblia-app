✦ ARTIGO 01

Introdução

A RedStar desenvolveu o aplicativo Bíblia Online com respeito integral à privacidade de seus usuários. Esta Política explica de forma clara quais informações são processadas, como são utilizadas e quais são os seus direitos, em conformidade com a Lei Geral de Proteção de Dados — LGPD (Lei nº 13.709/2018) e com as políticas do Google AdMob.

RESUMO DIRETO A RedStar não coleta nem armazena dados pessoais em servidores próprios. O único serviço de terceiros utilizado é o Google AdMob para exibição de anúncios.
✦ ARTIGO 02

Informações Coletadas

O que a RedStar NÃO coleta

A RedStar não coleta, armazena nem processa em seus próprios servidores nenhum dado pessoal identificável. O Aplicativo não exige criação de conta, login, e-mail ou qualquer cadastro.

Dados salvos localmente no seu dispositivo

As seguintes informações são salvas exclusivamente no armazenamento interno do seu dispositivo, nunca enviadas a qualquer servidor:

  • Versículos marcados como favoritos
  • Capítulos bíblicos salvos para leitura offline
  • Preferências de uso do Aplicativo

Esses dados são removidos automaticamente ao desinstalar o Aplicativo.

Dados coletados pelo Google AdMob

O Aplicativo utiliza o Google AdMob para exibir anúncios e manter o serviço gratuito. O SDK do AdMob pode coletar:

  • Identificador de publicidade do dispositivo (Advertising ID)
  • Endereço IP
  • Informações do dispositivo (modelo, sistema operacional, idioma)
  • Dados de interação com anúncios (cliques, impressões)
  • Localização aproximada baseada no IP
  • Dados de diagnóstico e desempenho do SDK

Esses dados são coletados e gerenciados exclusivamente pela Google LLC, conforme a política disponível em: policies.google.com/privacy

✦ ARTIGO 03

Finalidade do Uso dos Dados

Os dados coletados pelo Google AdMob são utilizados exclusivamente para:

  • Exibir anúncios relevantes e personalizados
  • Medir o desempenho dos anúncios
  • Prevenir fraudes publicitárias
  • Cumprir obrigações legais da Google

A RedStar não tem acesso aos dados individuais coletados pelo AdMob e não os utiliza para nenhuma outra finalidade.

✦ ARTIGO 04

Compartilhamento de Dados

A RedStar não vende, aluga, cede nem compartilha dados pessoais dos usuários com terceiros, exceto pelo funcionamento técnico do SDK do Google AdMob conforme descrito nesta política.

✦ ARTIGO 05

Segurança dos Dados

Os dados salvos localmente (favoritos, capítulos offline) ficam protegidos pelas salvaguardas de segurança do sistema operacional Android. A RedStar não tem acesso a esses dados.

✦ ARTIGO 06

Seus Direitos (LGPD)

Em conformidade com a LGPD, você tem direito a:

  • Acesso — saber quais dados seus estão sendo processados
  • Correção — solicitar correção de dados incompletos ou inexatos
  • Exclusão — remover dados locais desinstalando o Aplicativo
  • Portabilidade — solicitar seus dados em formato estruturado
  • Oposição — opor-se à publicidade personalizada
  • Revogação — revogar consentimento a qualquer momento
CANCELAR ANÚNCIOS PERSONALIZADOS Acesse Configurações → Google → Anúncios → "Cancelar personalização de anúncios" no seu dispositivo Android.
✦ ARTIGO 07

Retenção de Dados

Os dados armazenados localmente permanecem no dispositivo até que o usuário os remova manualmente ou desinstale o Aplicativo. A RedStar não retém nenhum dado em servidores próprios.

✦ ARTIGO 08

Transferência Internacional

O Google AdMob pode transferir dados para servidores fora do Brasil, incluindo os Estados Unidos. Essa transferência é realizada em conformidade com as salvaguardas exigidas pela LGPD e pelo GDPR europeu.

✦ ARTIGO 09

Crianças e Adolescentes

O Aplicativo é destinado ao público geral e adequado para todas as idades. Não coletamos intencionalmente dados de crianças menores de 13 anos além do que é tecnicamente processado pelo SDK do AdMob. Em caso de identificação de coleta não intencional, tomaremos as medidas necessárias para exclusão imediata.

✦ ARTIGO 10

Alterações nesta Política

Esta Política pode ser atualizada periodicamente. Alterações significativas serão comunicadas por aviso no próprio Aplicativo. A data de atualização será sempre indicada no topo deste documento.
